Profile Dropdown Test

Test Profile Dropdown Functionality

This test will verify that the profile dropdown works correctly when a user is signed in.

Expected Behavior

  • Not Signed In: Profile button shows "Profile", clicking opens sign-in modal
  • Signed In: Profile button shows user's name, clicking opens dropdown menu
  • Dropdown Menu: Shows user info, Edit Profile button, and other options
  • Edit Profile: Clicking opens the profile modal to edit age, weight, gender, etc.
